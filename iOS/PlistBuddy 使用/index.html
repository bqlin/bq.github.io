PlistBuddy 使用

发表于 2017-09-17 | 分类于 iOS开发

Plist文件是iOS项目中比较普遍的一种文件格式，类似于XML，都是通过键值对的形式存储数据，而PlistBuddy则是Mac自带的专门解析plist的友好工具。如果有自动化打包，动态修改iOS工程项目中的info.plist文件的需求，会用到PlistBuddy。

PlistBuddy 文档

输出

PlistBuddy -c ‘print’

容器类型取值作为键

:<键>:<索引>:<键>
以此类推。

更改

PlistBuddy -c ‘Set <键> <值类型> <值>’

删除

PlistBuddy -c ‘Delete <键>’

添加新值

PlistBuddy -c ‘Add :Version string 1.0’ info.plist

添加普通字段

PlistBuddy -c ‘Add <键> <值类型> <值>’

添加数组字段

PlistBuddy -c ‘Add :<数组名称> array’
PlistBuddy -c ‘Add :<数组名称>: <值类型> <值>’

添加字典字段

PlistBuddy -c ‘Add :<字典名称> dict’ info.plist
PlistBuddy -c ‘Add :<字典名称>:<字典键> <值类型> <值>’

合并 A 至 B

PlistBuddy -c ‘Merge ‘

B 与 A 重复的字段，会取 B 的值，即不替换，只增加新字段。
